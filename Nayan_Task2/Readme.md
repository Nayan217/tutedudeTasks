# Laundry Wallah Website

Task 2 - Created a simple laundry service website with services, pricing table, and booking form.

## What's in this project?

This is a basic website for a laundry service that includes:
- List of laundry services offered
- Pricing table for different services
- Contact form for booking
- Simple styling with CSS

## How to run

Open the `index.html` file in your web browser.

## What I learned

- Creating lists and tables in HTML
- Building forms with proper labels
- Adding basic CSS for styling
- Using images in HTML
