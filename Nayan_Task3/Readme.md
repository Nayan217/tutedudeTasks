# CSS Selectors Practice

Task 3 - Learning how to use different CSS selectors to style HTML elements.

## What is this?

This project shows how to use CSS selectors like classes, IDs, and special selectors like nth-child. I styled different elements without using inline CSS!

## How to run

Just open `index.html` in your browser. Make sure the style.css file is in the same folder.

## Concepts I learned

- **Class selectors** (.head) - can be used on multiple elements
- **ID selectors** (#bglime) - only for one element
- **Element selectors** (p, h1) - targets all elements of that type
- **Child selectors** (ul li) - targets elements inside other elements  
- **nth-child selector** - targets specific child like first, second, third etc
- **Hover effects** - changes style when mouse goes over it
- **Transitions** - makes changes smooth instead of instant
