# Laundry Services Hero Section

This is a website hero section for a laundry service. It has a navigation bar at the top and a main section with text and an image.

## What is this?

A simple landing page design with:
- Navigation bar with logo, menu, and username
- Hero section with heading, description, and call-to-action button
- Image on the right side
- Hover effects on links and button

## How to run

Open `index.html` in your browser. Make sure the `style.css` file is in the same folder and the `laundry.jpg` in `images` folder in the same folder as `index.html`

## What I learned

- Creating a **navbar** with logo, menu, and user section
- Using **inline-block** to place elements side by side
- Making a **hero section** with text on left and image on right
- Adding **hover effects** - links change color, button moves up slightly
- Using **transitions** to make hover effects smooth
- Using **viewport units** (vw, vh) for responsive sizing
