# CSS Box Model & Units Practice

Task 4 - Learning about CSS units (vh, vw), box model, and responsive design.

## What is this?

This project has a simple card with an image and button. I learned how to use viewport units to make things responsive (they adjust to screen size automatically).

## How to run

Open the `index.html` file in your browser. The image and button should fit nicely on any screen size because of viewport units!

## What I learned

- **Viewport units** (vh and vw) - they change based on screen size
  - vh = viewport height (100vh = full height)
  - vw = viewport width (100vw = full width)
- **Box-sizing: border-box** - makes width/height calculations easier
- **Google Fonts** - used Poppins font from Google
- **addEventListener** - better way to handle button clicks instead of onclick
- **Hover effects** - button grows and gets brighter on hover
- **Transitions** - makes hover effect smooth instead of sudden
