# Task 6 - Laundry Login Page

This is my basic login page project made using HTML and CSS only.
Main idea was to make split screen login design using flexbox.

## Files
- `index.html`
- `index.css`
- `Readme.md`

## Project Description
Page has 2 sides:
- Left side has laundry image.
- Right side has login form with email, password, login button and links.

I used full screen center layout and kept code simple.

## Concepts Demonstrated
- Flexbox (`display: flex`, `justify-content`, `align-items`)
- Child selector (`.right-panel > p`)
- Group selector (`.left-panel, .right-panel`)
- Pseudo classes (`:focus`, `:hover`)
- Media query for mobile view

## How to Run
1. Open folder `Nayan_Task6`.
2. Open `index.html` in browser (or Live Server).
3. Make sure `index.css` is in the same directory.
